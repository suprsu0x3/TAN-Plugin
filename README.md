# TAN-Plugin
Lightweight plugin for Starkiller (or PS-Empire) that sends Telegram notifications whenever a new agent connects â€” so you never miss a host coming online.

## Install
Prerequisites:
- Empire >=6.0.0

1. Install via Starkiller plugin marketplace
2. `poetry add twilio`

## Usage
From the [Twilio Console](https://www.twilio.com/console), get your Account SID, authentication token, and phone number.
Add them to the plugin, and set enabled to `True`!
